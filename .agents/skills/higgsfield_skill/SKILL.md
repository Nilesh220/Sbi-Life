# Higgsfield Skill (AI Video Creative Direction)

This skill helps configure cinematic prompting, script translation, and directorial parameters when working with the Higgsfield AI video pipeline.

## Prompts Structure Guidelines
1. **Scene Description Blueprint**:
   - Structure prompt as: `[Subject] in [Environment] performing [Action]. [Camera Movement], [Lighting/Style], [Film Grade].`
2. **Camera Directives**:
   - Use dynamic movements: `dolly zoom`, `cinematic pan`, `slow-motion drone tracking shot`, `low-angle perspective`, `rack focus`.
3. **Lighting & Atmosphere**:
   - Specify moods: `golden hour glow`, `volumetric cinematic lighting`, `neon cyberpunk ambient light`, `soft rim lighting`.
4. **Higgsfield Controls**:
   - Define character motion weight: high dynamic movement vs. static expressions.
   - Keep video prompts under 60 words for maximum text-adherence.
